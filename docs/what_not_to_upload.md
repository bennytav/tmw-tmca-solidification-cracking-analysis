# Files not recommended for public upload

Do not upload the full raw working folder as-is. Keep these private unless the supervisor decides otherwise:

- complete XML controller logs;
- CT volume files;
- SEM folders;
- original specimen-photo folders;
- intermediate plotting/figure-preparation files;
- old duplicate scripts;
- local Overleaf/PDF working files;
- files containing private local paths or personal information.

The public repository should contain the processed data, templates, code, documentation, and raw-data inventory.
